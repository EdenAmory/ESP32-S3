# ESP32-S3 Embedded System

**Nama:** Nathaniel Eden Amory  
**NRP:** 3223600015  
**Mata Kuliah:** Embedded System  
**Deskripsi:** Implementasi dan pengujian Input/Output (I/O) pada ESP32-S3 menggunakan FreeRTOS (Multi-Tasking) untuk 8 komponen I/O Menggunakan Wokwi.


## Komponen I/O yang Diuji (1-8)

1.  **LED** (Output) - Pin 2
2.  **BUZZER** (Output) - Pin 46
3.  **BUTTON A & B** (Input) - Pin 35, 19
4.  **POTENSIOMETER (POT)** (Input Analog) - Pin 5
5.  **OLED Display** (Output) - Pin 38 (SDA), 39 (SCL)
6.  **ENCODER** (Input) - Pin 41 (CLK), 40 (DT)
7.  **SERVO Motor** (Output) - Pin 18
8.  **STEPPER Motor** (Output) - Pin 4, 7, 8, 9
